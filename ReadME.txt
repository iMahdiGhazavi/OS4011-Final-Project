In order to run the project, after going to the xv6-public directory:
1. run "make qemu" in the terminal in order to compile the xv6 terminal and bring its terminal up.
2. Now among all the files, enter "ps" to run this file.

Output: The "ps" program, first creats some processes in the xv6 OS and allocates some memory to each of them, finally it shows all the running or runnable process with their PIDs sorted!
